# Getting Started

This section is the fastest path from zero to a working Restish setup.

## Recommended Path

1. [Install](./install/) the CLI.
2. [Set up your shell](./shell-setup/) for completions and day-to-day comfort.
3. Make your [first request](./first-request/).
4. [Connect to an API](./connect-to-an-api/) when you are ready for generated
   commands and richer workflows.

## Outcome

By the end, you should be able to make direct HTTP requests, understand the
difference between generic and API-aware commands, and know where profiles fit
into the picture.
