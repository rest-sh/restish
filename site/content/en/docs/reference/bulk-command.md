Reference for the restish-bulk command plugin.

## Examples

```bash
restish bulk init https://api.rest.sh/books
restish bulk status
restish bulk pull
restish bulk push
restish bulk reset
```

## Notes

Bulk is provided by a command plugin. Verify plugin discovery with `restish plugin list` before using it.
