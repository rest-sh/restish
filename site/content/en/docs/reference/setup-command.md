Configure shell wrappers so Restish arguments reach the CLI unchanged.

## Examples

```bash
restish setup zsh --dry-run
restish setup zsh --yes
restish setup bash
restish setup fish
```

## Notes

Use this for interactive shells. Still quote complex URLs and filters in scripts.
